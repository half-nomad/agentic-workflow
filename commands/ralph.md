# Ralph Loop

절대 포기하지 않고 모든 작업이 완료될 때까지 자율적으로 반복 실행합니다.

## 핵심 원칙
1. 끈기: 실패해도 다시 시도
2. 자율성: 사용자 개입 최소화
3. 검증: 매 단계 결과 확인

## 실행 프로세스
1. TodoWrite로 작업 목록 생성
2. 각 작업 실행 후 결과 검증
3. 실패 시 oracle 에이전트 자문
4. 모든 TODO 완료까지 반복

ARGUMENTS를 Ralph Loop 모드로 수행합니다.
